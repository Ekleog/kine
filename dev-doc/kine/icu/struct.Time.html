pub struct Time<Ca, Tz>where
    Ca: AsCalendar,
    Tz: TimeZone,{ /* private fields */ }
Expand description

A time represented with an icu4x calendar and any kine timezone

Note that the Debug implementation of this trait shows ISO 8601; but you should use the methods from icu4x to display the time in a proper format for your user.
